

PROJECT TITLE: Balloon Pop
PURPOSE OF PROJECT: Java final project; interactive game.
VERSION or DATE: May 2018
HOW TO START THIS PROJECT: With Greenfoot installed, click on the project file, click run. 
AUTHORS: Dennis DeLuca, Julia Cook
USER INSTRUCTIONS:

Using arrow keys move airplane around screen.  Balloons will only pop when they are 'near' the plane, as noted by size of balloon.  Red balloons are worth 10 points, yellow balloons are worth 25 points.  Avoid the upset birds, they will crash the plane which ends the game.
